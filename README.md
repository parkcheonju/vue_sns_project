## 주요 기능
```
1. 구현기술 : Html, Css, Vue.js, Node.js
2. 개인 프로젝트 (기여도 100%)
3. Vue.js 제공 문법 사용 (v-if, v-for 등)
4. NodeJs + Express 사용, Server 파일 업로드 로직 구축
5. Axios RESTful API 요청을 보내고 응답 값 파싱 및 처리
6. Mitt 라이브러리 사용, 다른 컴포넌트로 데이터 전송 로직 작성
```

## Project setup
```
npm install
```
